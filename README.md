# nodejs-api-design

### Useful Prisma CLI Commands

```bash
# Install Prisma and other dependencies
$ npm i typescript ts-node @types/node prisma --save-dev

# Initialize Prisma
$ npx prisma init

# Format the schema
$ npx prisma format

# Install Prisma client
$ npm i --save @prisma/client

# Migration
$ npx prisma migrate dev --name init

# Look the data
$ npx prisma studio
```
